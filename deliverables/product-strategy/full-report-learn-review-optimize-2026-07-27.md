## 1. 三路研究发现

### 1.1 认知科学研究（瑞思 · 用户研究员）

**3-6岁儿童核心认知特征**：

| 特征 | 含义 | 设计启示 |
|------|------|----------|
| 再认先于回忆 | 能认出字≠能独立写出 | 先强化再认，回忆为辅 |
| 工作记忆仅3-5项 | 一次教太多=无效 | 单次3新字+2复习字 |
| 正字法意识由阅读驱动 | 不应显性教偏旁 | 上下文暴露偏旁含义，不单独教 |
| SDT关联感效应量最大(g=1.776) | 看到字→想到关联含义 | 释义必须第一时间展示 |

**教学流程设计原则**：
- 渐进释放：展示释义→再认练习→描红书写→跟读产出
- 错误处理：不直接给答案，渐进提示（偏旁语义→部分拼音→完整答案）
- 单字学习时长：2-3分钟，符合注意力跨度

## 2. 产品需求概览（PRD核心）

> 完整PRD见：`deliverables/product-strategy/prd-learn-review-optimize-2026-07-27.md`

### 2.1 四步递进教学

```
展示释义(15s) → 再认练习(30s) → 描红书写(45s) → 跟读产出(30s)
    ↓               ↓               ↓               ↓
  建立关联感      巩固再认        精细编码        深度提取
```

- 单字2-3分钟，单次5-8分钟（3新字+2复习字）
- 每步都有明确的通过/未通过判定标准
- 未通过时进入渐进提示流程（偏旁语义→部分拼音→完整答案）

### 2.2 Leitner Box间隔重复

| Box | 复习间隔 | 升级条件 | 降级条件 |
|-----|---------|---------|---------|
| Box 1 | 1天 | 连续2次正确 | - |
| Box 2 | 3天 | 连续2次正确 | 1次错误→回Box1 |
| Box 3 | 7天 | 连续2次正确 | 1次错误→回Box1 |
| Box 4 | 14天 | 连续2次正确 | 1次错误→回Box1 |
| Box 5 | 30天 | - | 1次错误→回Box1 |

- 每日复习量 = min(今日到期数, 15字)
- 新字学习量 = 8 - 实际复习字数（保证总时长5-8分钟）

### 2.3 五级掌握状态机

```
new → seeing → familiar → mastered → solid
      (1次)    (3次)      (5次多维度)   (Box5)
```

- 旧"已掌握"映射为familiar（56%假阳性，不可信为mastered）
- mastered需5次不同维度验证（再认+书写+读音≥3维度）
- solid = 进入Box5，30天未遗忘

|--------|------|--------|----------|

| P1 | 6项 | 6人天 | Leitner Box + 四步递进 + 五级掌握 + 数据迁移 + review_logs读取 + "3+2"模式 |
