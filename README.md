# 儿童识字

面向 3-6 岁幼儿园儿童的汉字学习微信小程序，核心目标教会 2256 个常用汉字。

**当前版本：V2.0.0 — 愉悦体验全面升级**

## 功能

- **学习页**：大字卡片 + 拼音 + 百度 TTS 发音 + 百度 ASR 跟读识别，连续 3 次正确即掌握
- **复习页**：听音选字 / 看字说音双模式，连击系统（3连🔥 / 5连🔥🔥 / 7连🔥🔥🔥）
- **首页**：时段问候 + 数字滚动动画 + 成就徽章脉冲 + 学习/复习快捷入口
- **个人中心**：头像昵称 + 数据统计 + 成就展示 + 已掌握汉字列表
- **兴趣激励**：星星/小红花奖励 + 7级成就体系
- **V2.0 愉悦体验**：触感震动 + 弹性入场 + 水波纹 + 星星粒子 + 烟花庆祝 + 14组关键帧动画

## 技术栈

- 微信小程序原生开发（WXML / WXSS / JS）
- 微信云开发（云函数 + 云数据库 + 云存储）
- 百度语音 API（TTS 语音合成 + ASR 语音识别）
- ES5 兼容语法

## 项目结构

```
├── pages/                  # 6个页面
│   ├── index/              # 首页
│   ├── learn/              # 学习页
│   ├── review/             # 复习页
│   ├── profile/            # 个人中心
│   ├── mastered/           # 已掌握列表
│   └── settings/           # 设置页
├── utils/
│   └── delight.js          # V2.0 愉悦引擎（震动/动画/粒子/连击/文案）
├── cloudfunctions/
│   ├── login/              # 微信登录（获取 openid）
│   ├── main/               # 主业务逻辑（14个 action）
│   ├── fixData/            # 数据修复
│   └── import_chardata/    # 汉字数据导入
├── docs/                   # 产品文档
└── images/                 # TabBar 图标
```
